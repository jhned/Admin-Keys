Admin Keys provide intuitive admin keyboard shortcuts for accessibility and efficiency

== Description ==
Admin Keys provide keyboard interface for common tasks while managing your WP sites such as: switching back and forth from Admin to your public-facing website and vice versa; minimizing admin left menu sidebar for a wider and cleaner view, etc. A lot more are in the works so stay tuned! Suggestions and feature requests are most welcome!

The plugin has been tested extensively with WP_DEBUG enabled and works on multi-site network setup (see FAQ).

The plugin utilizes Mousetrap for handling key bindings.

For questions, suggestions or issues with this plugin, visit the plugin support page or contact me at wp [dot] joesat [at] gmail [dot ]com.

== Installation ==

1. Upload 'admin-keys' folder to '/wp-content/plugins/' directory

2. Activate the plugin through 'Plugins' menu in WordPress Admin

3. You're good to go! 

== Frequently Asked Questions ==

* Does this plugin work with multi-site setup?

Yes, but not extensively. Please report all inconsistencies/bugs found and we'll work it out asap.

* Was did plugin tested with debug option enabled?

Yes.

* Where is the admin page located?

In the admin panel, go to Settings -> Admin Keys. Right now, it's really NOT a settings page but more of an FAQ page with the list of the shortcuts.

* What are the shortcut keys?

** Toggle admin sidebar and top admin bar

** Redirects page from admin to homepage and vice versa

** Check all checkboxes

** Go to Add New Page

** Add New Post
